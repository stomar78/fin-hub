# SLOs and SLAs

## SLOs (Internal)
- Free valuation endpoint p95 ≤ 5s
- Error rate ≤ 1% over 1h window
- Cache hit ratio ≥ 70% for market/reference data

## SLAs (External, Phase 2+)
- Vetted report delivery within 72h from complete intake
- Support response within 1 business day

## Error Budgets
- Monthly error budget aligned to 99.0% availability for API
