# Contributing

## Dev Environment (to be added in Phase 1)
- Python 3.11 with venv, Node 20, Docker

## Conventions
- Python: black/ruff, type hints, pytest
- JS/TS: eslint/prettier, strict types
- Commits: Conventional Commits

## Branching & PRs
- Feature branches → PR → CI gates → squash merge

## Code Review
- Tests required for new logic; docs updated when behavior changes
