# Public Modules: Content Plan

- **User Dashboard (Saved Reports)**
- **API Marketplace**
- **Multi-Currency Interface**
- **Partner Portal Dashboard**

## User Dashboard (Saved Reports)
- **Purpose**
  Empower users to view, manage, and analyze saved valuation reports.
- **Sections**
  - Hero banner with headline, subtext, CTA.
  - Summary widgets: totals, accuracy, in-progress, last downloaded.
  - Interactive charts: valuations over time, category distribution.
  - Smart recommendations with AI prompts.
  - CTA strip for certified reports.

## API Marketplace
- **Purpose**
  Provide plug-and-play valuation APIs.
- **Sections**
  - Intro block with headline and CTA to docs.
  - API cards (endpoint, description, pricing, Try Now).
  - Technical highlights (encryption, SDKs, JSON outputs, compliance).
  - Developer resources (sandbox, explorer, webhooks).
  - Testimonial strip.

## Multi-Currency Interface
- **Purpose**
  Calculate valuations and pricing in preferred currencies with live FX.
- **Sections**
  - Hero message with CTA.
  - Currency selector (USD/AED/GBP/INR).
  - Pricing conversion block with live updates.
  - Informational note and SEO keywords.

## Partner Portal Dashboard
- **Purpose**
  Give valuation partners a secure, collaborative workspace.
- **Sections**
  - Hero banner and CTA.
  - Partner profile management.
  - Revenue dashboard and growth charts.
  - Collaboration center and partner tiers.
  - SEO keywords.
